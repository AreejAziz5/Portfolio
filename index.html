Project 1

Loan Default Prediction

The source: ( https://www.kaggle.com/datasets/kmldas/loan-default-prediction/data)

Tools: Python, Pandas, NumPy, Matplotlib, colab Notebook Description: Analyzed a loan default dataset to identify factors that influence whether a client is likely to default on a loan. The project focused on data cleaning, exploratory data analysis, and building a predictive model to support risk assessment decisions. Key Insights / Outcome: • Cleaned and preprocessed financial and demographic data • Performed exploratory data analysis to identify key risk indicators • Built and evaluated a prediction model to classify loan default cases • Provided insights that can support data-driven credit risk decisions

Conclusion: The results indicate that employment status and income level are key factors in predicting loan default. These insights can support financial institutions in improving credit risk assessment and making data-driven lending decisions.
